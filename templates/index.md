# Index

System manifest. What's where.

## Quick reference

| Need | Location |
|------|----------|
| Who I am | `me/profile.md` |
| How AI should behave | `me/preferences.md` |
| Where my thinking is | `me/thinking.md` |
| Current projects | `work/` |
| Ideas and seeds | `ideas/` |
| To-dos | `tasks/tasks.md` |
| Processing workflows | `skills/` |

## Folders

### `me/` — Identity
- `profile.md` — Role, background, key facts
- `preferences.md` — Communication style, AI behavior rules
- `thinking.md` — Frontier thinking: what I'm working on, what's connecting, open questions

### `work/` — Current context
- Active projects, domain intelligence, strategy docs
- Organized by project or topic

### `ideas/` — Idea capture
- Seeds, connections, things that don't have a home yet
- Status lifecycle: seed → developing → ready → incorporated → stale

### `reference/` — Published and external material
- Things I've written, articles I've read, external documents

### `inbox/` — Unprocessed inputs
- Raw material waiting to be processed and integrated

### `tasks/` — To-dos
- `tasks.md` — Active task list by priority

### `skills/` — Processing workflows
- How to handle different types of inputs
- Reusable procedures the AI follows

### `meetings/` — Meeting notes
- Processed transcripts and notes (not raw transcripts)

### `drafts/` — Work in progress

### `archive/` — Completed items
