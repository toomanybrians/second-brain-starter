# Skills Index

Skills are procedures the AI follows. When you ask to do something, the AI checks here first.

| Skill | Trigger Phrases | File |
|-------|----------------|------|
| Process Inbox | "process inbox", "what's in inbox" | `skills/process-inbox.md` |
| Integrate Knowledge | "integrate this", "wire this in" | `skills/integrate-knowledge.md` |

## Adding new skills

When you teach the AI a new workflow:
1. Offer to create a skill file
2. Document: trigger phrases, inputs needed, step-by-step procedure, output format
3. Add to this index
